# My Knowledge

Personal knowledge site built with VitePress and deployed with GitHub Pages.

## Commands

```bash
pnpm install
pnpm docs:dev
pnpm docs:build
pnpm docs:preview
```

## Content

- Add long-form articles under `docs/blog/`.
- Add topic notes under the matching `docs/*/` topic directory.
- Store local static assets under `docs/public/`.
