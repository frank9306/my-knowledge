## 我的小项目

下面是我自己维护的开源项目，方便有需要的同学直接使用或参考。

- [codex-switch-helper](https://github.com/frank9306/codex-switch-helper): Windows 上的 Codex App 桌面管理助手，基于 Tauri 构建。支持一键切换多个 Codex Home 配置（账号登录 / API Key 两种 Profile），自动写入用户级 `CODEX_HOME` 与 `OPENAI_API_KEY` 环境变量，并通过 `shell:AppsFolder` 直接启动 Codex App，也可一键恢复默认 Home。
- [learning-project-coach-skill](https://github.com/frank9306/learning-project-coach-skill): 把任意领域变成可持续推进的 30 天项目制学习仓库的 Agent Skill，覆盖领域地图、每日计划、作业检查、阶段测试、薄弱点诊断与最终 Demo。
- [quick-nav-extension](https://github.com/frank9306/quick-nav-extension): 一个现代化浏览器新标签页扩展，用个人导航站替换默认新标签页，支持卡片管理、智能搜索、批量编辑、每日备忘和 Bing 壁纸。
- [text-fragment-copier](https://github.com/frank9306/text-fragment-copier): 极简 Chrome 扩展，选中文本右键即可复制带 Text Fragment (`#:~:text=`) 的精确定位链接，分享时直接跳到原文。
