# AI Agent 学习资源整理

## 1\. AI Agent 学习路径

**链接**: https://zhuanlan.zhihu.com/p/1991174641620906427

这篇文章系统地介绍了 AI Agent 的学习路径，包括基础概念、核心技术栈、开发框架选择等内容。文章强调从商用场景出发，先掌握 AI 能力再学习具体工具，并提供了阶段式的学习建议和实战项目推荐。

## 2\. Lilian Weng 最新万字长文：揭秘大模型「思考」的奥秘

**链接**: https://zhuanlan.zhihu.com/p/1907237397969764822

前 OpenAI 应用 AI 研究负责人 Lilian Weng 的最新研究，深入探讨了大模型的推理能力。文章从 Chain-of-Thought (CoT) 到"顿悟时刻"(Aha Moment)，解析了测试时计算(Test-Time Compute)如何让模型变得更聪明，以及 DeepSeek-R1 等模型通过纯强化学习展现出的反思和回溯能力。

## 3\. Hello Agents

**链接**: https://datawhalechina.github.io/hello-agents/#/

Datawhale 开源的中文友好 AI Agent 入门教程，适合初学者系统地了解 Agent 的基础概念、工作原理和实现方式。

## 4\. AI Agents 设计模式

**链接**: https://adp.xindoo.xyz/

《Agentic Design Patterns》的中文翻译项目，系统介绍了 AI Agent 系统的 21 个核心设计模式，包括：

*   基础模式：提示词链、路由、并行化、反思、工具使用、规划
*   协作与记忆：多 Agent 协作、记忆管理、学习适应
*   高级能力：MCP协议、目标监控、异常处理、人机协同、RAG、Agent间通信等
*   生产实践：安全模式、评估监控、优先级排序、探索发现

还包含 7 个附录章节，涵盖高级提示技术、Agent 框架概览、编程 Agent 等深入内容。
